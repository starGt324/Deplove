4-Linux 101 hacks:

"Linux 101 Hacks" by Ramesh Natarajan is a comprehensive guide to mastering Linux command-line tools and system administration tasks. The book presents 101 "hacks," short, practical examples demonstrating various commands and techniques. It covers fundamental commands like cd, grep, find, sort, awk, and sed, along with more advanced topics like SSH configuration, date manipulation, shell scripting (including bash for loops and debugging), process management (ps, top, kill, nice), system monitoring (free, vmstat, netstat, sar, sysctl), package management (using yum, rpm, and apt), file archiving and compression (zip, tar, cpio, gzip, bzip2), partitioning and formatting disks (fdisk, parted, mkfs), setting up a LAMP stack (Apache, MySQL, PHP) including SSL, using XAMPP, and basic system security. Each "hack" provides clear instructions and output examples, making it easy for beginners to understand and apply the concepts. The book promotes efficient use of the command line for improved productivity. It also lists recommened books and additional online resources.
